# How to start

When the user says "start building" or "build the site":

1. Create `style.css` with all CSS variables, reset, and base styles first
2. Create `index.html` with the full page structure and placeholder content
3. Build section by section in the order listed in the page sections rules
4. After all sections are in place, implement the scroll-spy navigation
5. Do a final pass for responsiveness at all three breakpoints
6. Report any decisions made that deviated from this spec
