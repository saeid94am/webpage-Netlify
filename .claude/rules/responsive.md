# Responsive design

The site must work well at three breakpoints:

- **Desktop**: ≥ 1024px — full layout as described
- **Tablet**: 640px–1023px — same layout, slightly reduced font sizes
- **Mobile**: < 640px — navigation collapses to a hamburger menu,
  skills grid goes to 1 column, card padding reduces to 16px

Use CSS custom properties and media queries only. No CSS frameworks.
