# Page sections

Build the page in this exact order, top to bottom:

1. **Hero** — full name, title, one-line research focus, GitHub + LinkedIn links
2. **About** — 2–3 paragraph professional summary, research interests
3. **Experience** — work and research positions, reverse chronological
4. **Education** — degrees, institutions, years, thesis topics if relevant
5. **Research & Publications** — papers, conference contributions, preprints
6. **Projects** — key ML/medical AI projects with brief descriptions
7. **Selected Courses** — certifications and continuing education with links
8. **Skills & Tech Stack** — languages, frameworks, tools, medical domains
8. **Contact** — email, GitHub, LinkedIn, optional contact note

Each section must have:
- A visible `id` attribute matching its name (e.g. `id="experience"`)
  so it can be deep-linked and navigation anchors work
- A small sticky top navigation bar with anchor links to each section
