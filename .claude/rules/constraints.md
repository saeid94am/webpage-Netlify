# Do not do

- No animations or transitions except a subtle `color` transition on links (150ms)
- No hero background images, gradients, or decorative shapes
- No dark mode (light mode only)
- No JavaScript libraries or CDN scripts (except Google Fonts)
- No cookie banners, analytics scripts, or tracking of any kind
- No `<table>` elements for layout — use CSS grid/flexbox only
- No inline styles — all styles go in style.css
- No placeholder stock photos
- Do not change the color palette defined in the design system
- Do not use font weights above 700 or below 400
- Do not add sections not listed in the page sections rules
